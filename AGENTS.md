# Project: web data management web数据管理

## 1. 项目目标

这是一个网络爬虫和数据分析的全栈任务，你需要完成\docs中实验要求.ppt文件中包含的三个任务。主要目标如下：

1. 选择合适的技术栈和爬虫框架。
2. 选择合适的网站，爬取数据，留待后面分析。
3. 选取存在反爬的网站，克服反爬障碍。
4. 制作web应用，展示数据的爬取和分析成果。

## 2. 详细要求

首先阅读\docs路径中的实验要求，了解这个项目的要求。

然后给出项目的计划，供我审阅和修改。

- **中文书写**：你编写的代码注释、实验报告、讲解文档、教程文档都需要用中文书写。以md文件格式给出。存放在项目根目录的\notes文件夹中。
- 根据\docs路径下的实验报告模板，完成实验报告。实验报告的要求在ppt中给出。务必按照模板和实验报告要求完成报告
- 完成实验1时，需要爬取得到规模较大、数据类型较复杂的数据。数据包含的字段应该至少有4个，方便后面从不同角度分析数据，增加项目的复杂度。
- 你应当交付的文档有：三个实验的实验报告，网络爬虫和web数据管理的入门教程。
